## About

I work mostly on Laravel-based internal systems, especially ERP and operational tools that help teams handle daily work with fewer hidden assumptions.

My work usually sits between software development, business process mapping, deployment, documentation, and production support. I like turning messy workflows into systems that are easier to understand, maintain, and improve.
